# 🌌 Acerca del proyecto

**Movieshark** es una plataforma multimedia desarrollada para ofrecer una experiencia moderna de exploración y visualización de películas.

El proyecto integra servicios externos y herramientas de optimización para proporcionar navegación rápida, manejo eficiente de contenido y soporte multimedia avanzado.

La aplicación permite:

- 🎬 Explorar películas
- ⚡ Navegación rápida
- 🌐 Integración con subtítulos
- 📡 Streaming multimedia
- 🚀 Optimización mediante Redis
- 📱 Diseño adaptable

# ⚙️ Configuración

## 🌐 Variables de entorno

Configura las siguientes variables antes de ejecutar el proyecto:

```env
MOVIESHARK_OPENSUBTITLE_USER=your_user
MOVIESHARK_OPENSUBTITLE_PASS=your_password
REDISCLOUD_URL=your_redis_url
```

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaMultimediaModerna.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaMultimediaModerna
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar aplicación

```bash
npm start
```
